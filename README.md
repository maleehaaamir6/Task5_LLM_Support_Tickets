This project implements an automated, AI-driven support ticket classification system leveraging a pre-trained Zero-Shot Large Language Model (facebook/bart-large-mnli) to dynamically analyze and route incoming customer service requests. By deploying zero-shot learning methodology, the pipeline bypasses traditional training requirements, evaluating semantic content on the fly to categorize unstructured text into predefined operational tags: Technical Issue, Billing/Refund Inquiry, Product Information, and Account Security. Developed within Google Colab using Hugging Face's transformers library, the architecture extracts real-time confidence scores for every ticket to ensure precise routing logic. The complete framework is seamlessly integrated into a production-ready interactive dashboard via the Gradio API, providing a web-based user interface and a shareable public URL for live testing and deployment evaluation.
